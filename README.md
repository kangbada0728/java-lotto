# 4단계 - 로또(수동)

로또 구매
- 수동 구매
  - 1장 - 1000원
  - 2장 - 2000원
  - 돈이 1000원 이하이면 구매 불가
- 자동 구매
  - 1장 - 1000원
  - 2장 - 2000원
  - 돈이 1000원 이하이면 구매 불가

- 로또 한장 생성
  - 6개의 번호로 이루어져 있음
  - 번호는 1이상 45 이상

- 당첨번호 생성
  - 당첨번호 6개 생성
  - 보너스볼 1개 생성

- 로또 한장에 대한 당첨결과 구하기
- 로또 전체에 대한 당첨결과 구하기
- 당첨 통계 구하기

돈 계산
- 더하기
- 빼기
  - 빼기 결과 음수 불가
- 나누기
- 곱하기
